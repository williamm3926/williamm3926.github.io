Projects & Papers

01

H.I.S.S - The heated and Intelligent Snake System

Robotics Capstone · Winter 2025]

Designed for the Detroit Zoo, this project developed a heated transport crate for a reticulated python that automates the weighing process using load cells. The crate uses heat, darkness, and a narrow entryway to naturally motivate the snake to enter, eliminating the need for multiple caretakers to manually carry and restrain the animal. Testing confirmed near-perfect weight accuracy and sufficient heat distribution throughout the enclosure.

02

DiseasedCNN

Deep Learning for Robot Perception · Winter 2025

This project developed two deep learning models, DiseasedCNN and DiseasedCNN-Lite, capable of identifying 17 common crop diseases across 14 species for use in agricultural robotics. Using a technique called knowledge distillation, the lightweight DiseasedCNN-Lite was trained to mimic the accuracy of its larger teacher model, achieving 87% classification accuracy while being 334 times smaller at just 390 kilobytes. The result is a model practical enough to deploy on agricultural robots and drones, where memory and computational power are tightly constrained.

03

WormBot

Soft Robotics · Winter 2025

This project designed a soft pneumatic pipe-crawling robot for the Robosoft In-pipe Locomotion competition, using three individually controlled kresling origami actuators to navigate through ridged tubing. The robot relied on friction-based locomotion and flexible 3D-printed feet to traverse straight sections, turns, and a flexible tube obstacle — successfully completing all higher-priority sections despite a mid-competition actuator leak. The project demonstrates strong potential for future applications in pipeline inspection, search-and-rescue, and miniaturized medical robotics.
